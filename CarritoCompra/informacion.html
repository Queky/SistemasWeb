Informacion sobre la tienda

Para realizar esta tienda se ha utilizado HTML + CSS3 + Javascript además del conocido framework Jquery. He utilizado Jquery por que creo que hoy en dia se utiliza muchísimo y es una herramienta que complementa Javascript con gran cantidad de funciones que tienen utilidad en todos los ámbitos. Además su carga es mínima, si se utiliza por ejemplo, el CDN de google.

Para realizar el carrito se ha creado una "clase" tipo carrito con distintas funciones que implementan sus diferentes funcionalidades. Después para introducir productos se crean objetos con diferentes valores y se introducen en un array de esta clase.

Una vez se termina la compra, si le damos al botón check-out vamos a la pagina con el formulario de pago. El precio total se envía en la barra de direcciones.

El formulario utiliza expresiones regulares para comprobar que los datos introducidos son validos.
